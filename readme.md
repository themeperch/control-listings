## Overview

**Control Listings** is the ultimate directory manager for efficient listing control. Seamlessly add listings to your Citykid WordPress directory or classifieds ads portal.

With a user-friendly interface, you can quickly create, edit, and organize listings right from the frontend. This plugin is fully compatible with the Citykid WordPress theme, allowing seamless integration with advanced directory features.

## Features

- **Multiple Views:** Grid, list, and map views for listings.  
- **Advanced Search:** Filter listings by location, price, category, and more.  
- **Frontend Submission:** Allow users to submit their own listings.  
- **Map Integration:** OpenStreetMap (OSM) support, no API key needed.  
- **Bootstrap 5 Compatibility:** Fully responsive and modern.  
- **Multiple Sorting Options:** Sort by relevance, price, rating, or listing age.  
- **Detailed Listing Cards:** Images, description, features, price, rating, social share, bookmark, and like options.  
- **Single Listing Features:**  
  - Image slider  
  - Title and listing age  
  - Location display  
  - Features and key features list  
  - Tags and categories  
  - Working days and hours  
  - Enquiry options (call-to-action buttons)  
  - Audio/video gallery  
  - Events  
  - FAQs  
  - Awards and recognitions  
  - Ratings & reviews

## How to Use the Plugin

1. After activation, a new **Listings** menu appears in your WordPress admin.  
2. Click **Add New** under Listings.  
3. Provide a title, description, and relevant listing details.  
4. Save or publish your listing.

## Installation

### Directly from WordPress

1. Go to **Plugins > Add New**  
2. Search for **Control Listings**  
3. Click **Install Now** and then **Activate**

### Manual Installation

1. Download the plugin package.  
2. Unzip and move the folder `control-listings` to `/wp-content/plugins/`  
3. Activate the plugin from the WordPress Plugins screen
